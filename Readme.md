# InsightX

InsightX is an AI-powered text summarization web application built using Flask and Python. It helps users generate concise summaries from long texts and documents quickly and efficiently.

## Features

- User Registration & Login
- AI Text Summarization
- Document Upload Support
- Summary History
- User Profile Management
- Admin Dashboard
- Offline AI Model Support

## Technologies Used

- Python
- Flask
- SQLite
- HTML
- CSS
- JavaScript
- Transformers

## Installation

1. Clone the repository
2. Install dependencies: pip install -r requirements.txt
3. Go to folder scripts and run download_model.py to download the offline model
4. Run the application: python app.py
5. Open: http://127.0.0.1:5000
